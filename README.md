# Vaquinha

## Descrição

Aplicação web de arrecadação coletiva ("vaquinha"). Usuários se cadastram, autenticam-se e podem criar campanhas com meta de valor; qualquer usuário autenticado vê todas as campanhas públicas e contribui com valores (simulados, sem gateway de pagamento real). O progresso de arrecadação e a lista de contribuintes de cada campanha são visíveis a todos os usuários, caracterizando o compartilhamento de dados entre usuários.

Ver `ARCHITECTURE.md` para o desenho técnico e `DECISIONS.md` para o porquê de cada escolha.
